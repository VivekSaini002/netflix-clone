This is fully backend logic of netflix-clone which is help to learning skills.
In this backend project we use fully jwt token authentication and spring security.
Using role based access security.

technologies :- Java,
                Java8 features,
                SpringBoot,
                Spring Security,
                Jwt token,
                Data Jpa (MySql),
                Using postman for test all APIs, 
                
Admin Features :- Validate the email and sending the email according to situations,
                  create, delete, update and private/public (toggle) movies,
                  toggle publish video (Draft/Published),
                  manage all videos,
                  send the email verification :- User verification, Password verification, Frogot password verification and more.
                  
User Features :-  Users are own managend their account (create, update, delete),
                  Watch movies,
                  Searching feature include users search movies by title and description,
                  Add to watchlist and remove from watchlist,
                  add favorites and remove.
